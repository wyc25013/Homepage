Some Course Projects

Network File Server

In this project, we group implemented a network file server using knowledge of socket programming, remote procedure call, hierarchical file system and secure communication. More specifically, we implemented fs_session() to start a new session for a client fs_create(),fs_delete() to create/delete file or directory fs_writedata(),fs_readdata() to write/read data to/from a file. We also used lock() to handle race condition between multiple clients.

ARRIES-Database Recovery

This project is about a database recovery algorithm known as Algorithm for Recovery and Isolation Exploiting Semantics(ARIES). A database crash is simulated from software level. We implemented functions for the three key phases of ARIES: analyze(), redo(),undo() along with datastructures: Log Record, Transation Table and Dirty Page Table.

MultiThread Programming

In this project, we group implemented a thread library containing lock(),unlock(),wait(),signal(),broadcast(),join() for multithread programming using. We used lower level library <interrupt>,<guard> to guarantee the atomicity in kernal codes. We also used getcontext(),makecontext(),swapcontext() to manipulate context of threads. Finally, we used our implemented thread library to support an SSTF(Shortest Seek Time First) disk scheduler.

External Pager

In this project, we implemented an external pager for allocating memories. We implemented a fault handler to handle read/write fault and page fault. When page fault happens, we used clock algorithm to evict a virtual page from physical memory.

Basic Database Design

We used ER diagram to design a basic database for networking like facebook, so we called it fakebook. After that we wrote queries to create tables, load and retrieve data with Oracle SQLplus. Also we used JDBC to embed SQL into JAVA and we observed how optimizer works with Postgres.
